# 🧳 Itinerary - Aplicación de Itinerario Personal

Aplicación web para itinerarios de viaje, diseñada para dispositivos móviles y optimizada para viajes en grupo.

## ✨ Características

- 📱 **Diseño mobile-first** - Optimizado para consulta en dispositivos móviles
- 📅 **Itinerario cronológico** - Visualización clara por días con eventos ordenados
- 🔄 **Expansión sin modales** - Los eventos se expanden en su lugar manteniendo el contexto
- 💰 **Módulo financiero** - Resumen ejecutivo de gastos con conversión de moneda
- 🌓 **Tema claro/oscuro** - Cambio instantáneo con persistencia
- 📍 **Enlaces a Google Maps** - Navegación directa a ubicaciones
- 🚀 **Carga progresiva** - Scroll infinito para itinerarios largos
- 💾 **Persistencia de estado** - Guarda posición de scroll y eventos expandidos

## 🛠️ Tecnologías

- **Framework**: [Astro](https://astro.build/) v4.0
- **Interactividad**: [Alpine.js](https://alpinejs.dev/) con plugin Persist
- **Estilos**: CSS Variables con soporte claro/oscuro
- **Íconos**: [Material Symbols](https://fonts.google.com/icons)
- **Hosting**: GitHub Pages (estático)
